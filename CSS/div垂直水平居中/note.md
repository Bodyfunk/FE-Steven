## 手写垂直水平居中

### 常见的垂直水平居中方法
1. position+margin负值的方法（宽高固定）
2. position+margin:auto(固定宽高)
3. display:table-cell vertical-align:middle（固定宽度）【不常用】
4. position+transform(不需要固定宽高)
5. flex (不需要固定宽高)
